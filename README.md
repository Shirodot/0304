# Empirical Observation of Time Complexity

This repository contains a Java program designed to empirically observe and compare the time complexities of various algorithms. This serves as an educational exercise to understand how different algorithms scale as the input size ($n$) increases.

## 📝 Assignment Overview

The goal of this assignment is to:
1. Implement several algorithms in Java.
2. Print their running time as the input size increases.
3. Observe whether the runtime behavior matches their theoretical Big-O notation.

### Algorithms Implemented
* **Linear Scan (Array Sum)**: Expected time complexity is **$O(n)$**
* **Merge Sort**: Expected time complexity is **$O(n \log n)$**
* **Insertion Sort**: Expected time complexity is **$O(n^2)$**

## 🚀 How to Run

1. Make sure you have the Java Development Kit (JDK) installed on your system.
2. Clone this repository or download the `TimeComplexityObservation.java` file.
3. Open your terminal or command prompt and navigate to the folder containing the file.
4. Compile the Java file:
   ```bash
   javac TimeComplexityObservation.java
